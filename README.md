# Responsive Pizza Landing Page

Welcome to the repository for the Responsive Pizza Landing Page project. This project demonstrates a simple, mobile-friendly webpage design for a pizza shop, with sections for Home, About, Menu, Services, Contact, and more.

## Project Overview

The project consists of an HTML page styled with CSS and powered by JavaScript. It includes multiple sections with interactive elements, such as navigation links, call-to-action buttons, and social media icons. The page also features a responsive design that adjusts to various screen sizes.

### Technologies Used
- HTML
- CSS
- JavaScript
- Boxicons 
- External image sources 

## Sections

1. Header
   - Contains the logo, navigation menu, and a dark mode toggle.
2. Home
   - Displays an introductory section with a large image and a call-to-action button to view the menu.
3. About
   - Provides information about the pizza shop, with a description and a button to learn more.
4. Menu
   - Lists some popular pizza items with images, prices, and a shopping cart icon.
5. Services
   - Describes the order process, including ordering, shipping, and delivery stages.
6. Connect
   - Encourages users to connect with the pizza shop.
7. Contact
   - Offers contact information, social media links, and quick links to other sections of the page.

## Usage

To view the page, open the `pizza_shop.html` file in a web browser. Ensure that the accompanying CSS (`pizza_shop.css`) and JavaScript (`pizza_shop.js`) files are in the same directory.
